大三想推出的活动

1.小组式的阅读群

解释：

每个人兴趣不同，需要不同，所以如果硬是为了某个目的，大家一起做，会没有乐趣和成就感。
建立阅读群的目的是，大家可以寻找恰好和自己有共同需求或者兴趣的同学，一起享受或者是啃一本书。

好处：有团队的感觉，可以帮助自己定期看书，通过看其他同学的分享，更容易理解一本书。

步骤：

A.有想看的书：想看一本书，想找人和你一起看。

B.发布招募：在班群里把你想看的书说一下，然后创一个群，邀请大家来参加，管理员负责置顶信息。

C.群内分享：依据看的书的性质而定：如果是小说，建议定期发看的进度，以及自己的评论；如果是论述类的，则每看一部分，建议概括一下那部分的主要信息，分享到群里;……

D.群主总结（建议）：把群成员分享过的信息，整理成文档，发给每个群成员，若是品相较好，可以分享到班群，大家一起看。

2.学习后的归纳和分享

解释：

我们距离考研或者是职业医师已经是越来越近了，很多科目我们已经学完。相信部分同学会在暑假的时候，利用部分时间，来回顾一下之前的知识。

当我们把所接受的知识再加工，并且用自己的方式分享出来的时候，知识才真正得到内化。

好处：

分享者强化对内容的理解，阅读者或听众可花较少的时间，理解书本的内容

方式

A.班级微信分享：看过一块知识，尝试将它整理出来，成文档，最好配以让大家理解的图，发给班级微信后台管理人员，呈现与班级微信号上，供其他人阅读。
……
B.QQ讨论组直播：用文字描述是一回事，有时讲出来就是另一回事了，可以将自己理解的某一块知识，制成ppt，提交给学委，发布直播时间和内容。

3. 上一项的广意版本

解释：

可供分享的不只是学习的知识，也可以是自己的经验或者是兴趣

write it down

然后班级微信负责分享出去。

提示

可供分享的内容举栗：

  1. 暑假见习的经历
  2. 学校留校的感想
  3. 好用的软件推荐
  4. 好玩地方的推荐

关于脂质代谢

发表于 2017-05-07 | 分类于 医学 | 阅读次数

胆固醇 cholesterol

  • 分类：游离胆固醇（FC）和细胞内胆固醇 p.s.总胆固醇TC 是他们的合称
  • 合成主要原料：乙酰辅酶A（乙酰CoA）；丙二酸单酰辅酶A还原酶（HMG-CoA 还原酶）；能量；还原氢之类的。
  • 合成场所：所有有线粒体的细胞都可以合成，在胞质合成，主要合成细胞是肝细胞。

  • 细胞内胆固醇含量主要是和HMG-COA还原酶活性有关：

    A.昼夜节律性，晚上多，白天少；

    B.受产物或者是原料的调节；

    C.激素：胰岛素，促进合成 皮质醇and胰高血糖素 抑制合成。

  • 去路：

肝内的胆固醇：A.形成胆汁酸排出（有利于脂质的乳化吸收）；B.作为激素合成的原料：肾上腺皮质激素，雌激素，睾酮，孕酮等。
肝外的胆固醇：A.作为细胞膜的合成原料 B.通过乳糜微粒或者VLDL途径被吸收 C.通过HDL途径，被逆向转运至肝脏。

甘油三脂 triglyeride

  • 组成：甘油和脂肪酸
  • 合成场所：肝，脂肪组织，小肠
  • 途径：甘油一酯，甘油二酯

客观看待进化论

进化论和创造论的冲突

进化论>达尔文主义

创造论>科学创造论

即生物的出现和发现是否是一个完全自然的过程，没有任何干预

达尔文主义的代表作：物种起源

物种起源的三个主要论点：

  • 生物并非永久不变
  • 利用进化论可以解释世界上几乎所有生物的来源
  • 适者生存

他是怎么样来说明进化论的

用实验育种来说明自然选择的能力

  • 突变（是不规则、无目的、随机遇（random）而产生的改变）（突变一般是有害的）

    达尔文的证明方法：

    当达尔文写《物种起源》一书时，他不能提供任何自然选择进化的好例子。因为当时还没有人去寻找观察，所以他只有借用一些人工选择的例子，如人工改进牲畜及稼穑，不断选种培育毛最长的羊，生蛋多的鸡等等……育种的人成就可观，可以改变动物和植物各种特征，甚至使现有的品种与最初的野生品种（wild ancestors）之间产生很大的区别。

用赘述来形成理论：

怎样定义一种动物是否适应环境，得看它能否产生足够多的后代。 但是表面上的优势，有时候并不能让这个物种产生多的后代，也就是说，我们之后看到结果了之后，才知道，适者生存。

这样看来，任何一种特征都可以是优点或缺点，因为必须考虑它的环境如何才能确定。若有飞翔的能力，不是显然的优势吗？达尔文学说却假设自然选择驱使非洲马德拉岛（Madeira）上的甲虫失去飞翔的能力：岛上的风特别猛烈，会飞的昆虫反而容易披风吹到大海里去。另外，人类有较大的脑，但也需要一个大脑壳来保护它，因此使母亲在生产的时候受到更大的痛苦，甚至连性命也会遭遇更大的危险

e.g.孔雀的尾巴，没有价值，但是它们还是存留了下来。

自然选择论只是推理的论据
1.所有生物必须繁殖（reproduce）；
2.所有生物都表现可遗传的变异（hereditary variation);
3.遗传变异对繁殖的影响也各有不同；
4.所以对繁殖有利的变异必定成功，对繁殖不利的变异必定失败，因此生物就产生改变。
反例：活化石们，经过很长的时间，还是没有发生改变的物种。

无论整个群体改变与否，甚至在趋向减种的途中，任何群体之中总有一部份生物个体能留下较多的后代。

大小突变

1.小突变对繁殖是否有利？为什么无关竟要的突变带来的效果也存留了下来。
即使只有百分之五的一个眼睛的结构，他又有什么用呢？
2.微小突变累计，以至于最后形成心动器官和种族的概率有多少？
数学家和拥护达尔文主义者的冲突？

B.大突变
当胚胎发育时的号令有轻微的改变，很可能就导致成长的个体上有很大的变化。（概率依旧是极微小的）

化石难题

缺少中间型

绝大多数生物化石的历史都包涵两个与渐进式的进化论有冲突的特点：

1.稳定：过去绝大多数的生物活在地上的时候都没有显出任何进化或退化的现象。多数物种在地层中出现时与它们消灭时的外形几乎是完全一样的。即使有外形的改变也都十分有限，并且没有显示进化的一定方向。

2.突然出现：世界各地调查的结果证实，任何物种并非由始祖逐渐改变而来；相反的，各种生物出现时都已经“全部完成”了。

丘脑-垂体-肾上腺轴

发表于 2017-03-16 | 分类于 医学 | 阅读次数

这是一张下丘脑-垂体-肾上腺轴图

GSH 促肾上腺激素释放激素
VP 血管升压素
ACTH 促肾上腺激素激素
GC 糖皮质激素
MC 盐皮质激素
AngⅡ血管紧张素

按照线路来讲

  1. 下丘脑对腺垂体的作用：由室旁核分泌GSH，VP，在腺垂体上有相应的受体，促使腺垂体分泌ACTH。GSH的分泌有周期性：晚上分泌多，白天分泌少，自然下游激素也是这样一个状态。
  2. 腺垂体对肾上腺皮质的最作用： ACTH可以促进肾上腺皮质细胞里面核酸和蛋白质的合成，也可以促进细胞分裂，从而达到让adrenal gland 释放更多的GC，并且维持肾上腺皮质的大小，如果ACTH不足，那么肾上腺就萎缩。
  3. 负反馈，GC多的时候，抑制下丘脑GSH神经元的活动和腺垂体ACTH细胞的活性，抑制腺垂体对GSH的敏感性
  4. 下丘脑自生的超短反馈
  5. ACTH 过多对GSH的抑制
  6. 血管紧张素可以促进球状带细胞生长，提高MC 合成酶活性。血钾高，会促进MC合成

tip：当病人服用过多GC然后想停用的时候，不能马上停用，因为GC的 反馈作用，让腺垂体ACTH很少分泌，如果停了，那么自己产生的GC会过少，应该满满停，或者注射ACTH 促进病人自己的肾上腺生长。

黄疸

发表于 2017-03-08 | 分类于 医学 | 阅读次数

jaundice 黄疸

由于胆红素（bilirubin）在组织中推及，引起的皮肤，眼结膜，黏膜变黄。一般是由肝病和胆道疾病引起的。

早在希波克拉底时期，就有对黄疸的论述了，在十九世纪，就有了对黄疸的分类了。由胆道阻塞引起的，和不由其引起的。直到二十世纪，人们阐明了胆红素的代谢过程以及相应的检查技术得到发展，才得以对大部分黄疸病例给出解释。

虽然技术先进了，但是仍然要以耐心仔细的筛查和合适的诊断和治疗手段为基础，来对一些疑似的病例进行区分。

胆红素的代谢和计量

代谢

胆红素是由血红蛋白代谢产生的。

正常情况下，一个健康成年人平均每天产生4mg/kg 0.05mmol/70kg的胆红素

bilirubin来源：

生理条件下，70%-80%胆红素由衰老的红细胞破坏而来。还有一部分来自于在肝内其它含有血红素结合蛋白的物质，比如过氧化氢酶，和细胞色素氧化酶。尽管血红素结合蛋白比如说肌红蛋白在其它组织中也有，但是产生胆红素占比很小，所以忽略。总结：胆红素主要由衰老的红细胞和肝内血红素结合蛋白的代谢产生。

bilirubin 产生方式

血红素通过血红素氧化酶，变成胆绿素，然后经胆绿素还原酶，变成胆红素，一般在脾的巨噬细胞内，骨髓，kupffer cell，和肝细胞内完成。

运输方式及转化排出

在血浆中和白蛋白非共价结合（非结合胆红素），运输到肝，经过生物转化，变成亲水性较好的结合胆红素，然后才可以和胆汁一起经过胆道到小肠，一小部分则再次由小肠吸收，进行肠肝循环，大部分在肠道中，转化为胆素原，再转化为胆素，经粪便排除。

不同的诊断

  • 胆红素生成过多
  • 肝处理胆红素能力的下降

大致可以分为：胆红素代谢紊乱，肝病，胆道堵塞。

未结合胆红素过多

1.胆红素生成过多
溶血；无效红细胞的生成；血肿的吸收；创伤后输入过多的库存血。
2.肝细胞对未结合胆红素的摄取能力减弱
药物的影响：利福平（查）；免疫抑制剂，环孢素A；
3.胆红素转化为结合胆红素量下降
吉尔伯特综合征；

血清中结合和未结合胆红素都很高
