describe the basic class of cells found in the Sentra nervous system (CNS)

  • Neuron
  • Neuralgia
  • Vascular endothelium

  • neuralgia

    • Function :
      • support
      • formation of neuron circuit and synaptic plasticity神经元环路的形成，以及突触的可塑性
      • 参与组成髓鞘make myelin，让他保持绝缘；
      • 血脑屏障的形成
      • 炎症反应
      • 瘢痕组织的形成。

    • 分类：

      • astrocyte星形胶质细胞，

        最先在灰质中找到。在细胞体，树突和突触周围。

        1. 帮助保持细胞外液的例子平衡
        2. 递质的处理
        3. 帮组形成突出和环路
        4. 血脑屏障的形成和大脑室管膜屏障的形成brain ependymal barrier.
        5. 神经细胞坏死后位置的填充

      • oligodendrocyte 少突胶质细胞

        1. 在白质里发现的
        2. 构成髓鞘
        3. 周围神经系统叫做：shiwangcell
        4. 抗原的呈递

        5. 在某些CNS疾病中，会受到免疫攻击，multiple sclerosis

          郎飞节 node of Lanvier
          电信号可以在这里重新生成，这里通道很多，产生动作电位很快。

      • microglial cell 小胶质细胞

        1. 是一种单核吞噬细胞
        2. 分为两种状态：休眠状态叫 ramified cell 网状细胞，激活后，变成阿米巴形态，运动状态。
        3. 分泌细胞因子，调控周围的炎症反应

      • glial stem cell 干细胞

        1. 在血管周围
        2. 可以分化成各种神经细胞
        3. 干细胞的特征他都有：增殖，自我更新，分化

      • oligodenrocyte precursor少突胶质细胞的前体

        存在于白质中，主要分化为少突胶质细胞，也可能分化为其他细胞
