少尿型

分期

  1. 少尿期
  2. 移行期
  3. 多尿期
  4. 恢复期

原因

  1. 肾前性（功能性）
  2. 肾性（器质性）
  3. 肾后性

并发症

  1. 少尿期

    A.水中毒。 尿排除变少 导致水潴留B.高钾血症，钾随尿排出变少C.氮质血症，同上D.代谢性酸中毒，体内固定酸排出变少

  2. 移行期 A.肾小官上皮细胞开始重生，但是功能不完善，仍会有少尿期的疾病

  3. 多尿期 A.低钾血症，尿液排出多 B.低钙血症

  4. 恢复期

临床表现

头痛（脑水肿），抽搐/心跳减慢/传导阻滞（高钾），呼吸加深加快/抑制中枢神经/乏力（代酸）
